-----------------------
ゲーム紹介
-----------------------
ゲームタイトル：リゾートのワナ

リゾートの建築者にとして、ホテルからビーチまで、カスタマーがここですべてのお金を喜んで使うようにするために、どのような方法を使用していますか？


-----------------------
基本操作
-----------------------
ウィンドウサイズ　横144ｘ縦40

方向キー　↑、↓、←、→でカーソルを移動します。
ENTER確認します。
レベルの中にESCでレベル選択画面に戻ります。


-----------------------
システムの紹介
-----------------------
カスタマーは自分のステータスに応じて、違い建築に入ります。
↑：カスタマーはこの建築を入った後、このステータスが追加します。
↓：カスタマーはこの建築を入った後、このステータスが消去します。

お金が使われず、ステータスが消去していなければ、ビーチに着くと失敗します。
お金が全て使われた、ステータスが消去していなければ、ビーチに着くと一つの星をもらいました。
お金が使われず、ステータスが消去していた、ビーチに着くと二つの星をもらいました。
お金が全て使われた、ステータスが消去していた、ビーチに着くと三つの星をもらいました。
